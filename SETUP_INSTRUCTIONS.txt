✅ STEP 1: EXTRACT ZIP FILE
─────────────────────────────────────────────────────────────────────────────

1. Cari file: trading-dashboard.zip
2. Klik kanan → Extract All (Windows) atau double-click (Mac)
3. Folder "trading-dashboard" akan terbuat

Sekarang Anda memiliki semua file yang dibutuhkan! ✓

═══════════════════════════════════════════════════════════════════════════════

✅ STEP 2: INSTALL DEPENDENCIES (Optional - untuk development lokal)
─────────────────────────────────────────────────────────────────────────────

Jika ingin test di local computer:

1. Install Node.js dari https://nodejs.org (pilih LTS)

2. Buka Command Prompt/Terminal di folder "trading-dashboard"

3. Jalankan:
   npm install

4. Untuk run lokal:
   npm start

5. Dashboard akan terbuka di http://localhost:3000

   6. ✅ SELESAI! Dapatkan URL online dalam 2 menit

🆘 JIKA ADA MASALAH
─────────────────────────────────────────────────────────────────────────────

1. "npm: command not found"
   → Install Node.js dari https://nodejs.org

2. "Failed to build" saat deploy
   → Check syntax error di code
   → Run "npm install" locally dulu

3. "Blank white page" di browser
   → Cek browser console (F12)
   → Check internet connection
   → Clear cache (Ctrl+Shift+Delete)

4. Design terlihat aneh di mobile
   → Refresh browser (Ctrl+F5)
   → Check zoom = 100%

Baca DEPLOYMENT_GUIDE.md untuk troubleshooting lengkap!
